Tic-Tac-Toe

Tic-tac-toe or noughts and crosses or Xs and Os, is an interactive game on base of paper-and-pancil game for two players, X and O, who take turns marking the spaces in a 3×3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

Install

The game doesn't require any additional installation. You can start to play by clicking on file “index.html”. Also, you can drag-and-drop it to your browser.
